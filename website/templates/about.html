How it works?

In order to make a graph you will need a csv file with data in it. If you don't have it you can also one of the given sample data files.

  1. Upload a csv file

  2. Pick a desired color for the graph

  3. Pick a graph type

  4. Click on the "Generate Graph" button

All of the graphs that you made will be saved in "My account page" where you can download them as a png file or delete them.
